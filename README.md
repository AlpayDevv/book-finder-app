# 📚 Book Finder App

A simple Node.js + Express + EJS project that allows users to search books from a local JSON file acting as a fake API.

---

## 🚀 Features

- Search books by title
- Filter results dynamically
- Local JSON-based API
- Express backend
- EJS templating
- Clean UI with CSS

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- EJS
- JavaScript (ES Modules)
- HTML / CSS
- JSON (local database)
